About Dynamic Programming

Dynamic Programming is an Algorithm design paradigm that is used when problems consist of a number of overlapping subproblems. Dynamic programming can be used to first solve the simplest of subproblems and efficiently build up solutions to larger subproblems using the solutions we already computed.

Algorithms developed with the paradigms listed above can be used to solve hugely imporant problems such as:

  • Finding the shortest route from one location to another
  • Finding the most efficient way to schedule tasks
  • Finding the level of similarity between two RNA sequencences
  • Determining the optimal combination of items to store in a bag
