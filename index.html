About This Page

This is a simple website designed to teach readers about a selection of useful algorithm design paradigms.

About Algorithms

An Algorithm is a series of step instructions designed to solve a specific problem. Algorithm design is a major topic in the fields of Mathematics and Computer Science. Many different problems in Computer Science have a number of key similarities that allow them to be solved with similar algorithms. Computer Scientists have developed a number of strategies that can be used to design algorithms for a variety of problems. The major algorithmic design strategies that this website will cover are:

Divide and Conquer
Solves by breaking up into discrete subproblems, solving and recombining
Dynamic Programming
Solves by solving trivial problems first and iteratively building up to final solution
Greed
Solves by always finding best answer up to a certain point up until the final solution is found
Deciding Which Paradigm to Use
Knowing which paradigm to use is just as important as knowing the paradigms themselves

Algorithms developed with the paradigms listed above can be used to solve hugely imporant problems such as:

  • Searching and sorting large amounts of data efficiently
  • Finding the most efficient way to schedule tasks
  • Finding the shortest route from one location to another
  • Keeping your information secure on the internet
  • Compressing data
  • Keeping the internet running smoothly through efficient network routing
